# Signal Processing — MATLAB

MATLAB-based signal processing projects completed as part of an Intro to DSP course at Ben-Gurion University.

## Projects

### Project 1 & 2 — Sampling, Reconstruction & Frequency Analysis

Analyzed signal behavior in both the time and frequency domains. Key topics covered:

- Sampling and reconstruction of continuous-time signals
- Aliasing scenarios and the Nyquist criterion
- Comparison of results across different sampling strategies
- Accuracy measurement and error analysis
- Frequency domain analysis using the Fourier Transform

## Tools

- MATLAB
- Signal analysis: FFT, time-domain plots, frequency-domain plots
